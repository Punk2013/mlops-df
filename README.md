# mlops-df

## Запуск
В директории `src/`:
```
docker-compose up --build
```
## Описание
Система замена лица на фотографии(deepfake). Пользователю предоставляется графический интерфейс, предусматривающий следующие сценарии использования:
- Создание профиля человека с загрузкой его фотографий :white_check_mark:
- Просмотр добавленных профилей человек :white_check_mark:
- Обучение модели на фотогафиях добавленного профиля человека `- на данный момент нет gui`
- Замена лица на загруженной пользователем фотографии на лицо человека, на чьём профиле была обучена модель `- на данный момент нет gui, плохое качество`
